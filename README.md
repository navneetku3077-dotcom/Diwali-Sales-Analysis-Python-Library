# 🪔🎆 Diwali Sales Analysis using Python🎇

## 📌 ``**Brief One-Line Summary**``
An Exploratory Data Analysis (EDA) project to analyze Diwali sales data using Python.

## 📖 ``**Overview**``
This project analyzes customer purchasing behavior during the Diwali festival using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

## ❓ ``**Problem Statement**``
The objective is to identify customer purchasing patterns and generate business insights from sales data.

## 📂 ``**Dataset**``
- Diwali Sales Data.csv

## 🛠 ``**Tools and Technologies Used**``
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code
- Jupyter Notebook

## 📊 ``**Methods**``
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization

## 📈 ``**Key Insights**``
- Female customers spend more.
- Age group 26–35 contributes the highest sales.
- Top-performing states and product categories identified.

### 📊``**Result & conclusion**``
---
The analysis revels valuable customer purchasing trends that helps business improve making strategies and increase sales and revenue.

### Key Conclusion
- Female customer spends more than male customers.
- Customer aged **26-35years** contribute the highest sales.
- Uttar pradesh ,Maharastra and Karnataka are top performing states.
- Food ,clothing and electronic are the most demanding products categories.
